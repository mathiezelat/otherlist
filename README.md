# otherlist

Es una Aplicación web de notas simple realizado en React JS

## Instalación 

```bash
npm i
```
```bash
npm start
```
```bash
npm build
```
## Uso

Se puede agregar, eliminar notas y marcar si una nota ya fue realizada y este se guarda en el localStorage.
